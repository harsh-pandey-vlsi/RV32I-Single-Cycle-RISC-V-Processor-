# RV32I-Single-Cycle-RISC-V-Processor-
32-bit RV32I Single-Cycle RISC-V processor implemented in synthesizable Verilog and verified using a self-checking SystemVerilog testbench. Supports arithmetic, logical, load/store, branch, and jump instructions with modular RTL design, waveform-based validation, and ISA-compliant datapath architecture.
